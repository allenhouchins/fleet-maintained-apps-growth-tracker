## Workflow

1. **Daily Update** (12:00 PM UTC):
   - `.github/workflows/update-data.yml` runs
   - Clones fleetdm/fleet repository
   - Analyzes Git history of `ee/maintained-apps/outputs/apps.json`
   - Generates `data/apps_growth.csv`
   - Generates `index.html` with embedded data
   - Commits and pushes changes

2. **Deployment**:
   - `.github/workflows/deploy-pages.yml` runs on push
   - Deploys `index.html` to GitHub Pages
   - Website automatically updates

## Files Explained

- **generate_data.py**: Main script that pulls data from fleetdm/fleet
- **generate_html.py**: Converts CSV to HTML with embedded Chart.js visualization
- **data/apps_growth.csv**: Time-series data (date, app_count, apps_added_since_previous)
- **index.html**: Self-contained HTML file with embedded data (no external CSV needed)
